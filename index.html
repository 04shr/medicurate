Automated Pill Dispenser

An innovative and technologically advanced solution designed to streamline medication management for individuals with diverse healthcare needs.

Explore Features Get Started

Key Features

Multiple Medication Forms

Dispenses various forms of medication including tablets, capsules, refrigerated drugs, and syrups.

Medication Reminders

Timely audio and visual alerts to ensure you never miss a dose again.

IoT Connectivity

Remote monitoring and control via mobile app, allowing caregivers to track medication schedules.

Temperature Control

Built-in Peltier module for refrigerated medications that require specific storage conditions.

Key Components

ESP32 Microcontroller

Powerful dual-core processing with built-in Wi-Fi and Bluetooth connectivity for IoT capabilities.

Servomotors

Precise control over angular position for accurate medication dispensing from each slot.

3D Printed Slots

Custom-designed PLA compartments for various medication types with precise measurements.

Peltier Module

Thermoelectric device that creates temperature differential for cooling medications.
